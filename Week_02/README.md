# 算法训练营第二周学习笔记

## 数据结构学习

### 1.哈希、集合(Hash、Set)

    查询时间复杂度 O(1)
    添加时间复杂度 O(1)
    删除时间复杂度 O(1)

    哈希冲突后: 查询时间复杂度 O(N), 添加时间复杂度 O(N), 删除时间复杂度 O(N)

### 2.树

#### 二叉树

    前序遍历(根,左,右)
    中序遍历(左,根,右)
    后序遍历(左,右,根)

#### 二叉搜索树

    左子树上所有结点的值均小于它的根结点的值,右子树上所有结点的值均大于它的根结点的值
    查询时间复杂度 O(logN)
    添加时间复杂度 O(logN)
    删除时间复杂度 O(logN)
